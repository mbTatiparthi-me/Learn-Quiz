List Categories

GET https://learn-quiz.herokuapp.com/restapi.php?api=categories

Parameters

ParameterValueDescriptionData Type
sortasc | ascOrder sort attribute ascending or descendingString
orderbycat-id |
category-name |
category-image |
random		Sort collection by object attributeString
cat-idLimit result set to items with more specific by `cat_id`Integer
category-nameLimit result set to items with more specific by `category_name`String
category-imageLimit result set to items with more specific by `category_image`String

Example

- https://learn-quiz.herokuapp.com/restapi.php?api=categories
- https://learn-quiz.herokuapp.com/restapi.php?api=categories&orderby=cat_id
- https://learn-quiz.herokuapp.com/restapi.php?api=categories&orderby=random
- https://learn-quiz.herokuapp.com/restapi.php?api=categories&orderby=cat_id&sort=desc



Retrieve a Category 

GET https://learn-quiz.herokuapp.com/restapi.php?api=categories&cat-id=<cat-id>

Parameters

ParameterValueDescriptionData Type
cat-idUnique identifier for the object and use random for random IDsInteger

Example

- https://learn-quiz.herokuapp.com/restapi.php?api=categories&cat-id=1
- https://learn-quiz.herokuapp.com/restapi.php?api=categories&cat-id=2
- https://learn-quiz.herokuapp.com/restapi.php?api=categories&cat-id=random
